# UniSchedule - Automated Timetable Management System

A responsive web application integrated with Firebase Firestore for handling automated schedules, multi-variable constraints, and user dashboards.

## Project Setup
1. Install the Firebase CLI globally: `npm install -g firebase-tools`
2. Log in to your Firebase Account: `firebase login`
3. Initialize your project inside this root directory: `firebase init firestore`
4. Deploy the security configurations: `firebase deploy --only firestore:rules`
